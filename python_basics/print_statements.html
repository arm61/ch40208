Print Statements¶

Until now, we have been using the built-in functionality of the Jupyter Notebook to “print” information from the code we are writing, such as the value of a variable or it’s type. You may have noticed that the result of the final line in a given Jupyter cell will be printed. For example, if you run the code below in your Notebook so should get the same string shown,

to_print = "Jupyter returns the information in the last line"

to_print

'Jupyter returns the information in the last line'

However, if you would like to get information from other lines of the cell (or from within a large program or function, it is necessary to be able to print from the code. This is where the print statement comes in, which in Python looks like this,

print("Hello World!")

Hello World!

Which will output the string Hello World! when run. Some of you may be aware of the context of printing Hello World! in the history of learning programming, it is typically the first thing someone learns when they are learning a new programming language.

print() is a function, that takes a single argument, the object that is to be printed. In the first code example, where the Jupyter Notebook functionality is use to show the text, when this is run in your Jupyter Notebook, you will notice that to the left of the text there is Out[1]: (the number vary). This indicates that this is the output of the above cell.

In the second code example above, where the print() function is used. There is no additional test output to the left, which identifies that this is output information from the code within the cell.

Any of the types can be printed with the print statement. Additionally, it is also possible to use the print statement to insert numerical values into a string. We can even prescribe how the number is written. For example, in the code below, the information between the curly brackets defines that a float (f) should be written with two (2) numbers following the decimal point (.).

ideal_gas_constant = 8.314

print("R is approximately {}".format(ideal_gas_constant))
print("R to 2 decimal places is {:.2f}".format(ideal_gas_constant))

R is approximately 8.314
R to 2 decimal places is 8.31

More information about the format function can be found online.
